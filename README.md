# Weather & Air Quality Dashboard (Power BI)

## Project Overview

This project presents an interactive Weather and Air Quality Dashboard developed using Microsoft Power BI.
The dashboard provides comprehensive insights into current weather conditions, multi-day forecasts, and air quality indicators for Indian cities, with a primary focus on Pune.

It is created for data visualization, business intelligence analysis, and portfolio demonstration purposes.
